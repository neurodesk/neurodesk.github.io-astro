## Neurodesk components

Flexible tools that work together or independently. Neurodesk makes it easy for beginners and experts to use biomedical imaging tools on desktop, HPC, web, and cloud.

<div class="component-grid not-content">
	<a class="component-card" href="https://neurodesk.org/getting-started/neurodesktop/">
		<h3>Neurodesktop</h3>
		<p class="component-card__desc">Complete virtual desktop environment with GUI applications, ready to use in your browser or locally.</p>
		<ul>
			<li>100+ pre-installed neuroimaging tools</li>
			<li>Familiar desktop interface for graphical applications</li>
			<li>Install Neurodesk once, all tools are ready to use</li>
		</ul>
		<div class="component-card__media">
			<img src="/components/neurodesktop.png" alt="Neurodesktop" />
		</div>
	</a>

	<a class="component-card" href="https://neurodesk.org/getting-started/neurocontainers/">
		<h3>Neurocontainers</h3>
		<p class="component-card__desc">Individual containerised tools you can use in your own pipelines and workflows.</p>
		<ul>
			<li>Transparent, version-controlled builds</li>
			<li>Use via Docker or Singularity</li>
			<li>Perfect for HPC and cloud</li>
		</ul>
		<div class="component-card__media">
			<img src="/components/neurocontainer.png" alt="Neurocontainer" />
		</div>
	</a>

	<a class="component-card" href="https://neurodesk.org/getting-started/neurocommand/">
		<h3>Neurocommand</h3>
		<p class="component-card__desc">Command-line tool manager that fetches and runs containers seamlessly.</p>
		<ul>
			<li>Simple module-based interface</li>
			<li>Integrates with existing workflows</li>
			<li>Accessible through Neurodesk for command-line workflows</li>
		</ul>
