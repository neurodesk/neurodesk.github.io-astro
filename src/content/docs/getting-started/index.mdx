The current stable release is **{jupyterNeurodeskVersion}**. We regularly update Neurodesktop for performance and software updates. Check the [release history](/overview/release-history) for past releases. To upgrade your container, replace the release number with your desired version. For replicable analyses, share the stable release number you used so others can recreate your environment.

For the actual command to run on your system, see the per-OS guides under [Neurodesktop](/getting-started/neurodesktop/).
