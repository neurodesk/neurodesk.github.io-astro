Claude Code is Anthropic's command-line coding agent. You can run it inside a Neurodesk JupyterHub terminal and authenticate with an Anthropic subscription.

**1. Start Claude in a terminal.**

```bash
claude
```

**2. Trust the folder when prompted.**

**3. Authenticate with your Anthropic subscription.**

Follow the login prompt and open the URL Claude prints in a browser.

:::caution
Claude outputs a long authentication URL on the terminal. The JupyterHub terminal can wrap this URL across multiple lines, which makes it invalid when pasted. Copy the URL into an editor and remove the line breaks before pasting it into the browser.
:::
